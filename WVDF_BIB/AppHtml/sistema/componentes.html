Grupos de botões

Exemplos

Duas opções básicas, com duas variações.

Botão agrupado único

Agrupe os botões .btn dentro de .btn-group.

  1. <div class="btn-group">
  2.   <button class="btn">1</button>
  3.   <button class="btn">2</button>
  4.   <button class="btn">3</button>
  5. </div>

Vários grupos de botões

Combine séries em <div class="btn-group"> dentro de <div class="btn-toolbar"> para grupos mais complexos.

  1. <div class="btn-toolbar">
  2.   <div class="btn-group">
  3.     ...
  4.   </div>
  5. </div>

Grupos de botões na vertical

Faça com que os botões apareçam empilhados verticalmente.

  1. <div class="btn-group btn-group-vertical">
  2.   ...
  3. </div>

Dropdowns dentro de botões agrupados

Atenção! Botões com dropdowns podem ser agrupados dentro de seus próprios .btn-group within a .btn-toolbar para uma renderização adequada.

Navegação: abas e listas

Abas básicas

Pegue a tag <ul> e adicione a tag .nav-tabs:

  • Home
  • Profile
  • Messages
  1. <ul class="nav nav-tabs">
  2.   <li class="active">
  3.     <a href="#">Home</a>
  4.   </li>
  5.   <li><a href="#">...</a></li>
  6.   <li><a href="#">...</a></li>
  7. </ul>

Estado desabilitado

Para alguns componentes (tabs, pills, or list), adicione .disabled para deixar os links em cinza e nenhum efeito ao sobrepor o mouse. Os links permanecerão clicáveis, a não ser que remova o atributo href . Uma alternativa, é implementar um JavaScript para prevenir que esses links sejam clicados.

  • Clickable link
  • Clickable link
  • Disabled link
  1. <ul class="nav nav-pills">
  2.   ...
  3.   <li class="disabled"><a href="#">Home</a></li>
  4.   ...
  5. </ul>

Alinhamento dos componentes

Para alinhas os links na navegação,utilize as classes.pull-left ou .pull-right . Ambas irão adicionar uma classe flutuante na direção especificada.

Empilhável

As abas e as pílulas são dispostas por padrão na horizontal, adicionando essa segunda classe: .nav-stacked, fará com que ela apareça verticalmente.

A barra de progressão utiliza grandientes com CSS3, transições e animações que não são suportados no IE7-9 ou versões antigas do Firefox.
